Loan Module Parameters

Module parameters define global configuration values that control how the loan system behaves.

Parameter List

Parameter Description
settlement_denom Token denomination used for loan settlement
min_loan_amount Minimum loan size allowed
max_loan_amount Maximum loan size allowed
max_tenor_months Maximum loan duration
laz_authorities Addresses allowed to approve or reject loans
omnibus_authorities Addresses allowed to confirm disbursement and repayment

Parameter Governance

Parameters can only be modified via governance proposals.

Example Proposal

{
  "title": "Update Loan Params",
  "deposit": "10000000stake",
  "messages": [
    {
      "@type": "/cosmos.loan.v1.MsgUpdateParams",
      "params": {
        "settlement_denom": "stake"
      }
    }
  ]
}
